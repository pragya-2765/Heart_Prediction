# ❤️ Heart Disease Prediction

A Machine Learning web application built using **Streamlit** that predicts the likelihood of heart disease based on user health parameters.

## 🚀 Features

- Predicts heart disease risk using a Logistic Regression model
- Simple and interactive Streamlit interface
- Fast and accurate predictions
- User-friendly input form
- Machine Learning based prediction

## 🛠️ Technologies Used

- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Joblib

## 📂 Project Structure

```
Heart_Prediction/

```

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/pragya-2765/Heart_Prediction.git
```

Install the required packages:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```

## 📸 Demo

The application allows users to enter medical parameters and predicts the possibility of heart disease using a trained Logistic Regression model.
